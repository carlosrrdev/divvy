# Divvy

Expense tracking and sharing application

> This app is currently under active construction

## Description

Divvy is a web application that allows users to track their expenses and easily
split and share those expenses between other users. With Divvy, you can split
your expenses however you want. Split all expenses evenly between parties, 
assign expenses per person, and generate detailed results that you can then 
share between your party members so each know exactly how much their share of
the expenses is.
